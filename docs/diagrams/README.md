## How to export PNGs

1. Open [diagrams.net](https://app.diagrams.net/)
2. **File → Open from → Device** and select a `.drawio` file from this folder
3. Tweak layout/colors if you like
4. **File → Export as → PNG**
5. Save to `docs/images/` using the filename in the table above
6. Commit the PNG files — they will render in the root `README.md`

## Tips

- Use **transparent background** off (white background reads better on GitHub)
- Scale: **200%** for crisp PNGs on retina displays
- Border width: **10px** optional padding
